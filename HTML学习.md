# HTML

## 01.初体验跑马灯与标签

<marquee>
	跑马灯跑起来
	<input>
</marquee>

## 02.标签属性

<marquee loop="1" bgcolor="green" id="1">
	跑马灯跑起来
	<input type="password">
</marquee>

## 03.html结构

<html>
  <head> 
   <title> 我的网页 </title> 
  </head>
  <body>
	<marquee loop="1" bgcolor="green" id="1">
	跑马灯跑起来
	<input type="password">
	</marquee >
   </body>
<input disabled>
</html>

## 04.注释

  </head>
